# 🎯 NEON SNIPER — Finger Gun AR  
> A premium browser-based webcam shooting game powered by real-time hand tracking.

---

## 🚀 Overview

**NEON SNIPER** is an immersive, single-page web game where you use your **index finger as a gun** to aim and shoot targets in real-time using your webcam.

It blends:
- 🎮 Arcade shooting mechanics  
- 🖐️ AI-powered hand tracking (MediaPipe)  
- ✨ Neon cyberpunk UI & effects  

No installations. No controllers. Just your hand.

---

## 🎮 Gameplay

### Controls
- 👉 **Point index finger** → Aim  
- 🤏 **Curl index finger** → Shoot  
- ⌨️ **Spacebar** → Shoot (fallback)  
- ⌨️ **R** → Reload  
- 🖱️ Mouse → Fallback aiming & shooting  

### Objective
Destroy as many targets as possible before time runs out.

### Mechanics
- 🔫 Ammo system with reload  
- 🔥 Combo multiplier (up to x8)  
- ⏱️ 60-second timed rounds  
- 🎯 Multiple target types with different speed & points
